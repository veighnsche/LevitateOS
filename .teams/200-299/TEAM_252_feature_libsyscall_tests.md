# Team 252: Libsyscall Comprehensive Tests

## Goal
Implement a comprehensive test suite for `libsyscall` to ensure full coverage of the syscall ABI, including error paths and edge cases.

## Feature
We will add new test binaries to `userspace/levbox/src/bin/test/` targeting:
1.  **Metadata/Links**: `stat_test`, `link_test`
2.  **Time**: `time_test`
3.  **Scheduling**: `sched_yield_test`
4.  **Error Paths**: `error_test` (negative testing)
